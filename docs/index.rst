FIT Classification: a Python library
************************************

Welcome to our project. **FIT Classification: a Python library**
is a Pyhton library that enables you to use API of the Classification project
from within your Python scripts and programs. It also takes care of
storing and updating an access token for you and simplifies some other things.
For more information please refer to the sections below.
